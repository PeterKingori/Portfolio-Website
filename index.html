Projects

My Car Maintenance

An Android app for users to record information about their vehicle and the details of regular service or tune-ups that are done. You can keep track of the maintenance work that has been done on your vehicle(s). All the data is saved on a database on the device and can be deleted at any time.

The app was built in Android Studio using Kotlin and Jetpack architecture components. The design pattern used was Model-View-ViewModel (MVVM).

Built using: Android Kotlin
Code Google Play Store

Draw This or That

A simple drawing app for people of all ages. You can select different brush sizes, different colours, upload an image and draw on top of it, and save and share your creations. All the data is saved on a database on the device and can be deleted at any time.

The app was built in Android Studio using Kotlin and Jetpack architecture components.

Built using: Android Kotlin
Code Google Play Store

Graham Financial News

A news website built for a fictional financial news organisation that has a print edition but no online presence. In addition to the news articles there is also a page for an online shop for visitors to buy previous and special editions of the newspaper.

The web app has been built using HTML, CSS and JavaScript, and fetches the articles from a REST api and presents them to the readers. Other sample articles are from International Finance and World Finance.

Built using: HTML CSS JavaScript
